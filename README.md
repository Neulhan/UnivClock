# 한양대학교 종강시계 Extension

```
"아... 개강한지 일주일 됐는데 벌써 종강이 기다려지는,,,"
당신을 위해 준비했습니다!!!

종강까지를 초 단위로 계산해주는 크롬 확장프로그램!!!

"한양대학교 종강시계"

크롬에서 새 탭을 열 때마다 종강까지 남은 시간을 확인하세요.

그 밖의 강력한 기능들을 확인하세요!!

- 배경화면 설정 기능
- 한양대학교 주요 사이트 링크
- 한양대학교 메인 학사공지 확인
- 오늘의 사근동 날씨 확인

한양대생을 위한 최고의 크롬 확장 프로그램과 지금 함께하세요!
```
